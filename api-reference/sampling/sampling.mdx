---
openapi: post /v1/email/verify/batch/sample
---

This endpoint will create special batch with sample of emails.
It will give fast feedback regarding deliverability/toxicity of uploaded emails.

**Rate Limiting**

Default rate limit is 200 requests per hour.

**Results**

After the initial POST request another endpoint can be used to check it's status/results.

```bash
curl --request GET \
  --url https://api.usebouncer.com/v1/email/verify/batch/sample/{id} \
  --header 'x-api-key: <x-api-key>'
```

Response JSON will look something like this:
